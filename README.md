# SmartCart Teppanku

Personal Shopping Assistant for Teppanyaki Catering

## Fitur

* Katalog Produk
* Shopping Cart
* Expense Analyzer
* Statistik Belanja
* Form Validasi
* Async Data Loading
* State Management menggunakan setState

## Teknologi

* Flutter
* Dart
* Material Design 3

## Cara Menjalankan

flutter pub get

flutter run

## Dibuat Untuk

UTS Pemrograman Mobile
